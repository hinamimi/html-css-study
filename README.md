# HTML/CSS 練習用レポジトリ

## 準備

### Windows

- WSL を入れる
- git 周りの設定

### MacOS

- brew を入れる

### 共通

- vscode をインストール
- anyenv をインストール
- nodenv をインストール
- node 19.3.0 をインストール
- npm ci
